# About

**ChronoGeoLab** is an open toolkit for time-geographic visualization, space-time
analysis, and mobility-based exposure research.

It is developed by **[GISPark Lab](https://juewang.space/#GISPARKLAB)**, built by
**Devin Yongzhao Wu** and **Jue Wang**.
